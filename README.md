# FanBridge

FanBridge is an AI-powered stadium navigation and incident management platform for FIFA World Cup 2026. It helps fans, volunteers, and stadium staff with real-time guidance, multilingual support, accessibility assistance, and operational intelligence.

## Live Demo

[FanBridge App](https://fanbridge.netlify.app/)

## Features

- AI-powered fan assistance
- Stadium navigation and wayfinding
- Multilingual chat support
- Accessibility-friendly interactions
- Incident reporting and response workflows
- Volunteer and operations support
- Stadium information and venue guidance

## Tech Stack

- Frontend: React + Vite
- Styling: Tailwind CSS
- Backend: Node.js + Express
- AI: Google Generative AI
